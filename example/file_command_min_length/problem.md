## 問題文

与えられたディレクトリ内の `.txt` ファイルを対象に、
`内容文字数が最も短いファイル`を見つけ、そのファイル名を標準出力に1行で出力してください。

- 同じ文字数が最短のファイルが複数ある場合は、いずれを出力しても可とします。
- 対象は `.txt` ファイルのみとし、サブディレクトリは含みません。

## 制約

- 対象ディレクトリには `.txt` ファイルが少なくとも1件存在します。
- 文字列は UTF-8 で保存されます。

## 入力

標準入力に対象ディレクトリのパスを1行渡されます。

## 出力

条件を満たすファイル名を1行で出力してください。

### 入力例1

`smallest` ディレクトリ内: `readme.txt`(5文字), `notes.txt`(9文字), `main.txt`(15文字)

```
smallest
```

出力例

```
readme.txt
```

### 入力例2

`symbols` ディレクトリ内: `short.txt`(1文字), `middle.txt`(2文字), `long.txt`(3文字)

```
symbols
```

出力例

```
short.txt
```
