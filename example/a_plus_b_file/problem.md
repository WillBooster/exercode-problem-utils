## 問題文

整数$A$が書かれたファイル`a.txt`と、整数$B$が書かれたファイル`b.txt`が与えられます。
$A+B$の計算結果をファイル`c.txt`に出力してください。

## 制約

- $0 \leq A,B \leq 10^9$
- 入力は全て整数である。

---

## 入力

入力は以下の形式で標準入力から与えられる。

- `a.txt`: 整数$A$が書かれている。
- `b.txt`: 整数$B$が書かれている。

---

## 出力

- `c.txt`: $A+B$を出力せよ。

---

## 入力例1

### a.txt

```
2
```

### b.txt

```
3
```

## 出力例1

### c.txt

```
5
```
